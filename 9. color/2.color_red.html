	#RRGGBB 								HEX코드값 단위
												RR-red, GG-green, BB-blue
												각각의 해당 값은 00 ~ FF까지 16진수로 표현됩니다.
							
	rgb(red, green, blue) 				RGB색상값 단위
												0~255 사이의 값으로 배합율은 0% ~ 100% 입니다.
												255인 경우 100% 배합율입니다.
												
	rgba(red, green, blue, alpha)	RGBA 색상값 단위
												0 ~ 255 사이의 값으로 배합율은 0% ~ 100% 입니다.
												255인 경우 100% 배합율입니다.
												alpha는 0.0 ~ 1.0으로 투명도 입니다. 1.0인 경우 불투명입니다.
												
	hsl(hue, saturation, lightness)	HSL 색상값 단위
												hue : 0 ~ 360도의 값을 가집니다. 색상값입니다.
												saturation : 0% ~ 100% 값을 가지며, 채도값입니다.
												lightness : 0% ~ 100% 값을 가지며, 밝기 명도 값입니다.
												
	hsla(hue, saturation, lightness, alpha) 			HSLA 색상값 단위
												
												hue : 0 ~ 360도의 값을 가집니다. 색상값입니다.
												saturation : 0% ~ 100% 값을 가지며, 채도값입니다.
												lightness : 0% ~ 100% 값을 가지며, 밝기 명도 값입니다.
												alpha는 0.0 ~ 1.0 으로 투명도입니다. 1.0인 경우 불투명입니다
