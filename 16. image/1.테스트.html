img 태그		html에 이미지를 표시해 줍니다.
	src	 	img태그의 속성으로 이미지 파일의 경로 및 파일명을 작성해줍니다.
	alt		img태그의 속성으로 이미지 파일을 못 보는(장애인) 경우에 설명글 입니다.
	width	img 태그의 속성으로 이미지 파일의 너비를 지정합니다. 픽셀, %를 사용할 수 있습니다.
	height	img 태그의 속성으로 이미지 파일의 높이를 지정합니다. 픽셀, %를 사용할 수 있습니다.
	
	img 태그에서 주의할 점은 이미지 파일의 경로 및 파일명.확장자가 정확해야 합니다.
	width 또는 height 중 하나의 값만을 지정하고 사용하는 경우가 많습니다.
	width 값만을 지정한 경우에는 height는 비례해서 변경이 됩니다.
