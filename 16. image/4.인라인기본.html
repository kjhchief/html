div는 기본적으로 block 요소입니다. block 요소는 html 문서의 한 가로공간을 다 차지하게 됩니다.

1. display: block; block 요소화 하였습니다. div태그의 기본값이므로 변화가 없습니다.
1. display: block; block 요소화 하였습니다. div태그의 기본값이므로 변화가 없습니다.
2. display: inline; inline으로 적용하여 요소가 칸,칸으로 배치되어집니다. inline값에 의하여 width와 height 속성이 무시됩니다.
2. display: inline; inline으로 적용하여 요소가 칸,칸으로 배치되어집니다. inline값에 의하여 width와 height 속성이 무시됩니다.
3. display: inline-block; block 속성과, inline속성을 가지게 됩니다. width와 height 속성 인식합니다.
3. display: inline-block; block 속성과, inline속성을 가지게 됩니다. width와 height 속성 인식합니다.
